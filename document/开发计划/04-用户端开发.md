# 用户端开发

1️⃣ 安装依赖

```bash
npm init vite@latest frontend -- --template vue
cd frontend
npm install axios vue-router@4
```

2️⃣ 项目结构

```bash
frontend/
├─ public/
│   └─ index.html
├─ src/
│   ├─ main.js
│   ├─ App.vue
│   ├─ router/
│   │   └─ index.js
│   ├─ pages/
│   │   ├─ Menu.vue
│   │   ├─ Cart.vue
│   │   ├─ Orders.vue
│   │   └─ Review.vue
│   ├─ components/
│   │   ├─ DishCard.vue
│   │   └─ CartItem.vue
│   └─ api/
│       └─ index.js
├─ package.json
└─ vite.config.js
```
