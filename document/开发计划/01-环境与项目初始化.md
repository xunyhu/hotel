# 环境与项目初始化

## 一、环境准备

1. 安装 Node.js (>=18)

    ```base
    node -v
    # 如果 < 18，去 https://nodejs.org 下载 LTS 版本
    ```

2. 安装 MySQL 8

    启动后设置 root 密码

    创建数据库：

    ```base
    port 3306
    password 123456
    admin admin
    CREATE DATABASE hotel_saas DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;
    ```

3. 安装 Redis 6

    ```base
    redis-server --version
    ```

4. 安装 Nginx（部署阶段使用，先安装即可）

    ```base
    nginx -v
    ```
