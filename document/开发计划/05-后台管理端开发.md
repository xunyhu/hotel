# 后台管理端开发

1️⃣ 项目初始化

```bash
npx create-react-app frontend-admin --template cra-template-pwa
cd frontend-admin

# 安装常用依赖
npm install axios react-router-dom@6 antd @ant-design/icons
npm install dayjs classnames
```

2️⃣ 项目目录结构

```bash
frontend-admin/
├─ public/
├─ src/
│  ├─ api/             # 封装接口
│  │   └─ index.js
│  ├─ assets/          # 图片、图标等资源
│  ├─ components/      # 公共组件，如 TableList、FormModal
│  ├─ pages/           # 页面级组件
│  │   ├─ Dashboard/   # 首页统计
│  │   ├─ Users/       # 用户管理
│  │   ├─ Roles/       # 角色权限管理
│  │   ├─ Dishes/      # 菜品管理
│  │   ├─ Orders/      # 订单管理
│  │   └─ Reviews/     # 评论管理
│  ├─ routes/          # 路由配置
│  │   └─ index.jsx
│  ├─ utils/           # 工具函数
│  ├─ App.jsx
│  └─ index.jsx
```
